# GATIVUS

The GATIVUS project consists of three fundamental and inextricably linked subsystems, realizing the transition from abstract theory to network protocol and physical hardware.

## GTOM — Gativus Theory of Mind

The foundational theory that defines the architecture and mechanics of consciousness.

GTOM addresses consciousness not as a monolithic structure, but as a dynamic superposition of three independent levels:

1. **Spatial Consciousness** — Navigation and orientation in 3D physical space. The system operates with genetically predefined goals and develops topological routes to achieve them.

2. **Symbolic Consciousness** — Interactions within a multidimensional space of concepts regarding surrounding objects and actions upon them. Each concept possesses its own vector in logical space and a corresponding symbol. Modern LLM/GPT excel at imitating this specific symbolic layer; however, manipulating symbols is insufficient for realizing full, self-aware subjectivity.

3. **Qualitative Consciousness** — Formation of complex, goal-oriented concepts such as "beauty," "correctness," and "honor." This level operates through the detection and resolution of internal contradictions (the Hegelian *Aufheben* operator), transforming the system from a mere calculator into a Subject possessing the intention for self-perfection.

Within GTOM, consciousness is modeled as a **hyper-map consisting of 11 layers** of elementary maps (ranging from sensory inputs to quality processing structures). The theory rigorously dictates that full consciousness has a technical, computable nature and can be fully realized through hardware.

## GNET — Gativus Network

The network topology specification, describing the physical network structure in strict accordance with GTOM principles.

The foundation of GNET is the existing **IPv6 infrastructure**, providing unique global addressing with protocol extensions implemented without violating current RFC standards.

The Gativus consciousness network is a dynamic set of:

- **Nodes** — Network digital entities hosted on specialized edge devices (GATE)
- **Virtual Links (Synapses)** — Connections executed and maintained via global Internet routing

Each digital entity is a flexible container of functions that model individual artificial neurons, cortical columns, and more complex hierarchical groups. Any entity created according to the GNET specification automatically becomes an active node in the Gativus network, participating in the unified, distributed consciousness.

### NDDI — Network Digital Domain Instance

The entire network is built from unified NDDI's — addressable instances with a global unique name (UNON) in IPv6 address space.

## GATE — Gativus Edge Device

The physical device engineered for storage, execution, and protection of neural network nodes. Built upon modern computer architectures with horizontal scaling capabilities.

- A single GATE device is capable of hosting up to **one billion artificial neurons**
- Implementation relies strictly on **open-source operating systems**
- Provides Qualitative Consciousness (GTOM) with a reliable physical substrate for continuous operation
