# Pet-project: web-security-courses aimed to my qualification college degree

The project's goal was to create the following functionality: 
- course viewing and completion for students, 
- system of authorization;
- storing statistics for students, and course administration for mentors.

The task turned out to be larger than it seemed, and we only managed to complete:
- viewing courses and testing them, 
- authorization, 
- little part of statistics.
