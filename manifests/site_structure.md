# Текущая структура сайта и системная навигация

Для анализа рефакторинга ниже представлена текущая карта страниц, названий и иконок проекта.

## Основная навигация (Header/Footer)

| Путь | Текущее название | Иконка (Heroicons) | Описание |
| :--- | :--- | :--- | :--- |
| `/` | `Главная` | `i-heroicons-home` | Главная страница приветствия |
| `/courses` | `Темы` / `Курсы` | `i-heroicons-book-open` | Список доступных курсов |
| `/profile` | `Профиль` | `i-heroicons-user` | Личный кабинет пользователя |
| `/login` | `Авторизация` | `i-heroicons-lock-closed` | Страница входа в систему |

## Доступные, но не выведенные в меню страницы

| Путь | Файл | Статус | Примечание |
| :--- | :--- | :--- | :--- |
| `/about` | `about.vue` | Активна | Информационная страница |
| `/admin` | `admin.vue` | Активна | Панель администратора |
| `/courses/[slug]` | `[...slug].vue` | Динамическая | Страница с самим контентом курса |

## Используемые компоненты интерфейса (UI)

- **theHeader.vue**: Основное верхнее меню (использует `UHorizontalNavigation`).
- **theFooter.vue**: Подвал сайта (дублирует навигацию).
- **theSiteLogo.vue**: Логотип сайта (отдельный компонент).
- **theThemeChanger.vue**: Переключатель темной/светлой темы.
- **theProfileIcon.vue**: Иконка профиля в хедере.

## Анализ имен и иконок
1. **Непоследовательность**: В хедере `/courses` называется «Темы», а в футере — «Курсы».
2. **Иконки**: Используется стандартный набор `i-heroicons-*`. Можно заменить на более специфичные для информационной безопасности или образовательной платформы.
3. **Admin**: Ссылки на админку нет в интерфейсе рядового пользователя.
