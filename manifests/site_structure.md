# Текущая структура сайта и системная навигация

В рамках создания энциклопедии «Gativus Wiki», ниже представлена актуальная карта страниц, названий и иконок проекта.

## Основная навигация (Header/Footer)

| Путь | Название | Иконка (Heroicons) | Описание |
| :--- | :--- | :--- | :--- |
| `/` | `Главная` | `i-heroicons-home` | Главная страница приветствия (Лендинг) |
| `/books` | `Библиотека` | `i-heroicons-building-library` | Раздел сборников/книг Gativus |
| `/articles` | `Статьи` | `i-heroicons-document-text` | Индекс всех статей и методологий |
| `/glossary` | `Глоссарий` | `i-heroicons-document-magnifying-glass` | Термины и определения проекта |

## Доступные, но не выведенные в меню страницы

| Путь | Файл | Статус | Примечание |
| :--- | :--- | :--- | :--- |
| `/about` | `about.vue` | Активна | Контактная информация / О проекте |
| `/login` | `login.vue` | Активна | Страница входа в систему Gativus |
| `/books/[slug]` | `[slug].vue` | Динамическая | Страница детализации конкретной книги и её глав |
| `/articles/[slug]` | `[slug].vue` | Динамическая | Страница чтения конкретной статьи/главы |

## Панель администратора (Только по ролям)

| Путь | Описание |
| :--- | :--- |
| `/admin` | Главная страница (Dashboard) панели администратора |
| `/admin/books` | Управление книгами (Индекс) |
| `/admin/books/create` | Создание новой книги |
| `/admin/books/[id]/edit` | Редактирование, привязка глав и обрезка обложки |
| `/admin/articles` | *Раздел статей* (API/Страницы в разработке или в планах) |

## Используемые компоненты интерфейса (UI)

- **theHeader.vue**: Основное верхнее меню (responsive, с адаптивным мобильным offcanvas).
- **theFooter.vue**: Подвал сайта.
- **theSiteLogo.vue**: Логотип сайта с градиентом Gativus.
- **theThemeChanger.vue**: Переключатель темной/светлой темы.
- **theProfileIcon.vue**: Иконка профиля в хедере и управление аккаунтом.
- **theToc.vue**: Липкое боковое оглавление для статей (`IntersectionObserver`).
- **thePresentationView.vue**: Модуль просмотра презентаций.
- **theLeftQuizSelector.vue**: Боковой переключатель "Теория/Презентация".
- **TheScrollToTop.vue**: Кнопка возврата наверх (появляется при прокрутке).

## Основные цвета и стили
- `sky-600` / `sky-400` как основной акцентный цвет для кнопок и ссылок.
- `zinc-900` как основной фон в темной теме вместо стандартного черного.
- Активное использование анимаций, градиентов и `backdrop-blur`.
