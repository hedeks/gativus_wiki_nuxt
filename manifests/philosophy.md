Книги - набор статей, которые объединены по смыслу. Каждая статья является 1 главой книги. Либо же книга как отдельный .odm файл.

Книга представляет собой набор ссылок на статьи. У книги есть название, категория, Описание, и картинка, соответствующая ей.

А у статьи в свою очередь есть текст статьи из набора заголовков, h1 заголовок только один, он же является названием статьи. У статей есть категория. И описание.

Далее идут термины, для каждого термина есть статья, в которой есть определение термина. У термина есть категория. И описание.

Категории для всех одинаковы, они характеризуют тематику статьи/книги/термина.

В итоге в идеальном раскладе по категории может быть написана книга или статья, а так же ассоциированы термины, если это все потом визуализировать в виде графа получится онтология знаний.
