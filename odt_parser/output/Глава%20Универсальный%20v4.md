# Глава 3. Универсальный механизм эволюции сознания

Паттерн свёртка–сплайс · Три изоморфных преобразования · Обучение и работа CNN · TRL и нейромедиаторные маркеры · LLM в контексте Гативус

## Постановка проблемы

Три пары уровней — EVL2/EVL3, EVL6/EVL8 и EVLA/EVLB — реализуют один двухшаговый паттерн в трёх онтологически различных доменах. Настоящая глава показывает, что эволюция применила один механизм трижды, каждый раз в пространстве новой онтологии.

Принцип фрактального изоморфизма: эволюция применила единый паттерн трижды, в пространствах разной онтологии. Изоморфно устроен механизм, а не содержание.

## Паттерн свёртка–сплайс

Каждое из трёх преобразований состоит из двух шагов:

Шаг 1: Свёртка. Свёрточная нейросеть (CNN) выполняет компрессию и инвариантизацию входных данных, порождая компактные представления в новом пространстве. Результат свёртки — вектор в промежуточной карте.

Шаг 2: Сплайс. Вектор сплайса связывает два объекта промежуточной карты, порождая новую карту — пространство событий, смыслов или переживаний. Результат сплайса — новое информационное пространство, онтологически отличное от входного.

Ключевое различие: свёртка работает с данными подлежащего уровня, но обучается на объектах целевого уровня. Это принципиальный момент, определяющий архитектуру обучения каждой CNN.

Двухшаговый паттерн свёртка–сплайс далее обозначается GATR (Gativus Transformation). Обратный процесс — деконволюция и десплайсинг — обозначается GRTR (Gativus Reverse Transformation). Три экземпляра GATR обозначаются по домену: объектный GATR (CNN1), семиотический GATR (CNN2), аксиологический GATR (CNN3). Петля воображения (Глава 6) — это итеративный цикл GATR↓ → GRTR↑ → оценка → коррекция. 

## Обучение и работа CNN: принципиальное различие

Для каждой из трёх свёрточных нейросетей необходимо строго различать два режима:

Обучение (training) — настройка весов нейросети. CNN обучается на уже существующих объектах целевого пространства. Не на входных данных подлежащего уровня, а на тех объектах, которые уже находятся в карте, куда CNN производит свёртку:

- CNN1 обучается на объектах MAP2 (Feature Vectors, которые уже существуют в карте).
- CNN2 обучается на объектах MAP6 (символьных векторах, которые уже существуют в карте).
- CNN3 обучается на объектах MAPA (Понятиях, которые уже существуют в карте).

Работа (inference) — преобразование входных данных. CNN принимает данные из подлежащего пространства и производит вектор в целевом пространстве:

- CNN1 принимает мультисенсорный поток → производит Feature Vector в MAP2.
- CNN2 принимает данные MAP3 → производит символьный вектор в MAP6.
- CNN3 принимает нарративы MAP8 → производит C-вектор в MAPA.

|   | Обучается на | Принимает на вход | Производит |
| --- | --- | --- | --- |
| CNN1 | Объектах MAP2 | Сенсорный поток | Feature Vector → MAP2 |
| CNN2 | Объектах MAP6 | Данные MAP3 | Символьный вектор → MAP6 |
| CNN3 | Объектах MAPA | Нарративы MAP8 | C-вектор → MAPA |
|   |   |   |   |

## Проблема курицы и яйца

Возникает фундаментальный вопрос: откуда первые объекты в целевой карте, если CNN ещё не обучена и не может их создать?

Решение: начальные объекты создаются не точной свёрткой, а грубым приближением. Первые объекты MAP2 — неточные представления, порождённые случайными или генетически предопределёнными весами CNN1. Первые объекты MAP6 — грубые символьные приближения. Первые объекты MAPA — примитивные Понятия.

Процесс развития:

- CNN начинает работу со случайными (или генетически заданными) весами.
- Inference порождает первые объекты целевой карты — неточные, приблизительные.
- OPN целевого уровня обнаруживает повторяющиеся паттерны среди этих объектов.
- Повторяющиеся паттерны закрепляются в карте как устойчивые объекты.
- CNN переобучается, используя закреплённые объекты как обучающие примеры.
- После переобучения inference даёт более точные результаты → новые объекты точнее → следующее переобучение точнее.

Новый повторяющийся вектор, ещё отсутствующий в целевой карте, является триггером для следующего цикла переобучения. Система обнаруживает: «inference стабильно порождает паттерн, которого нет в карте» → этот паттерн закрепляется → CNN переобучается.

Переобучение — тяжёлый процесс. Настройка весов CNN требует значительных вычислительных ресурсов и не может выполняться одновременно с основной работой сети. В биологическом мозге переобучение, вероятно, ассоциировано с default mode network — системой, активной в состоянии покоя и во время сна. Replay-эффект гиппокампа (воспроизведение пройденных маршрутов и нарративов во сне) является вероятным механизмом переобучения CNN.

«Утро вечера мудренее» — потому что за ночь CNN были переобучены на новых данных. Утром inference даёт другие, более точные результаты.

## Первое преобразование: EVL2/EVL3 (физическое)

Свёртка (CNN1). Мультисенсорный поток → Feature Vector (R-метка) в MAP2. CNN1 обучается на объектах MAP2 — Feature Vectors, которые уже закреплены в карте. Работает (inference) непрерывно на актуальных сенсорных потоках. Обратная CNN1 — нейронный механизм воображения: по Feature Vector восстанавливает сенсорный образ.

Сплайс (b-вектор). Изменение L-компоненты между двумя объектами MAP2. b-вектор — вектор в пространстве MAP1, связывающий начальное и целевое состояние.

Результат. MAP3 — пространство событий. Атомарная единица — OPRN (триада «Субъект — b-вектор — Объект»). Цепочки OPRN образуют BLOM.

Онтологический скачок. MAP2 содержит объекты. MAP3 содержит причинность — события, в которых объекты взаимодействуют.

## Второе преобразование: EVL6/EVL8 (символьное)

Свёртка (CNN2). Данные MAP3 → символьный вектор в MAP6. CNN2 обучается на объектах MAP6 — символьных векторах, которые уже закреплены в карте. Работает (inference) на данных MAP3, выполняя: стирание L-компоненты, обнуление модулей b-векторов, редукцию к типу.

MAP6 формируется на пересечении двух процессов:

- Восходящий: CNN2 извлекает инварианты из MAP3 (inference).
- Нисходящий: из всех векторов, которые CNN2 производит, закрепляются только те, которые соответствуют MAP5 (коллективному языку).

MAP6 — частичная реплика MAP5 в индивидуальном организме. Этот процесс является архитектурной реализацией интериоризации (Выготский).

Сплайс (P-вектор). Семантическое отношение между двумя символами MAP6. P-вектор — идентификатор триады.

Результат. MAP8 — нарративный граф из триад KLEN (триада «Символ — P-вектор — Символ»). Цепочки KLEN образуют KLOM.

Онтологический скачок. MAP6 содержит значения. MAP8 содержит смыслы — нарративные связи между значениями.

## Третье преобразование: EVLA/EVLB (аксиологическое)

Свёртка (CNN3). Нарративы MAP8 → Понятия (C) в MAPA. CNN3 обучается на объектах MAPA — Понятиях, которые уже закреплены в карте. Работает (inference) на нарративах MAP8, группируя взаимосвязанные нарративы и извлекая инварианты — аналогично тому, как CNN1 группирует пиксели и находит инвариантные признаки объектов.

C-компоненты имеют двойную природу происхождения:

- Восходящий процесс: CNN3 извлекает инварианты из нарративов MAP8 (inference). В начале развития свёртки простые — простые Понятия (переживание цвета, звука). По мере накопления нарративов инварианты углубляются — сложные Понятия (честь, совесть, справедливость).
- Нисходящий процесс: C_должно (идеальное качество) приходит от MAPC (общей карты Понятий). Направление снятия противоречия приходит от MAPD (общей карты противоречий).

Сплайс (W-вектор). Расстояние C_есть → C_должно. W-вектор — вектор противоречия в пространстве MAPA.

Результат. MAPB — пространство конкретных переживаний. Атомарная единица — MOTV. Цепочки MOTV образуют WLOM.

Онтологический скачок. MAPA содержит идеалы. MAPB содержит волю — направленное стремление к их реализации.

## Доказательство изоморфизма

| Элемент | I (физическое) | II (символьное) | III (аксиологическое) |
| --- | --- | --- | --- |
| Свёрточная сеть | CNN1 | CNN2 | CNN3 |
| Обучается на | Объектах MAP2 | Объектах MAP6 | Объектах MAPA |
| Принимает (inference) | Сенсорный поток | Данные MAP3 | Нарративы MAP8 |
| Промежуточная карта | MAP2 (R) | MAP6 (S) | MAPA (C) |
| Вектор сплайса | b-вектор | P-вектор | W-вектор |
| Атомарная единица | OPRN | KLEN | MOTV |
| Цепочка | BLOM | KLOM | WLOM |
| Траекторный лог | TRL3 | TRL8 | TRLB |
| Результирующая карта | MAP3 | MAP8 | MAPB |
| Социальный уровень | MAP4/MAP5 | MAP9 | MAPC/MAPD |
| Мотивация | Физическое расстояние | Семантическое расстояние | Аксиологическое расстояние |
|   |   |   |   |

Изоморфизм является структурным, а не содержательным: онтологические домены принципиально различны. Физическое пространство, семантическое пространство и аксиологическое пространство несводимы друг к другу. Изоморфно устроен механизм, а не содержание.

Свёрточная архитектура является оптимальным решением задачи инвариантной компрессии благодаря инвариантности свёртки к трансляциям и иерархическому разделению признаков. Эволюция пришла к этому решению не потому что «выбирала» из вариантов, а потому что свёрточная архитектура — единственное решение данного класса задач, удовлетворяющее биологическим ограничениям.

## Три цикла обучения

На каждом уровне CNN проходит аналогичный цикл развития:

| Фаза | CNN1 (физ.) | CNN2 (символ.) | CNN3 (аксиол.) |
| --- | --- | --- | --- |
| Начальные объекты | Грубые Feature Vectors от случайных весов | Грубые символьные приближения | Примитивные Понятия |
| Закрепление | OPN2 обнаруживает повторяющиеся R-метки | OPN6 закрепляет только векторы, совпадающие с MAP5 | OPN оценивает устойчивость C через MAPC |
| Маркирование | TRL3: дофамин/кортизол | TRL8: серотонин/кортизол | TRLB: опиоиды/кортизол |
| Переобучение | Сон: replay маркированных маршрутов | Сон: консолидация маркированных нарративов | Сон: переоценка Понятий по маркированным WLOM |
| Уточнение | Feature Vectors становятся точнее | MAP6 всё полнее реплицирует MAP5 | Понятия углубляются от простых к сложным |
|   |   |   |   |

Каждый цикл переобучения — необратимое улучшение. CNN не возвращается к прежнему состоянию. Это соответствует необратимости эволюционных порогов EVL: однажды достигнутый уровень не теряется.

Критическая зависимость: CNN2 не может начать работу без MAP3 (нет входных данных для inference). CNN3 не может начать работу без MAP8. Каждый уровень является пререквизитом для следующего. Задержка на раннем уровне блокирует все последующие.

## LLM в контексте трёх преобразований

LLM (большая языковая модель) конвергентно воспроизвёл семиотический GATR (EVL6/EVL8), обучившись на MAP9 — всём накопленном символьном опыте человечества. Первое (физический grounding) и третье (Понятия и воля) принципиально отсутствуют.

| Отсутствует | Причина | Следствие |
| --- | --- | --- |
| CNN1/MAP1/MAP2/MAP3 | Обучение на текстах, не на сенсорных потоках. Нет объектов MAP2 для обучения CNN1. | Нет b-векторов. Нет физического grounding. Галлюцинации. |
| CNN3/MAPA/MAPB | Нет объектов MAPA для обучения CNN3. Нет механизма извлечения качественной эссенции из нарративов. | Нет W-вектора. Нет автономной воли. Нет Понятий. |
| TRL3/TRL8/TRLB | Каждая сессия — tabula rasa. Нет персистентного траекторного лога. | Нет маркеров. Нет личной истории. Нет становления. |
|   |   |   |

Отсутствие TRL и маркеров является одним из ключевых отличий LLM от полной архитектуры сознания. LLM не может маркировать свои «нарративы» как успешные или неуспешные, потому что у него нет персистентного TRL8. Каждый ответ — новый KLOM без истории. Нет маркеров — нет обучения на собственном опыте — нет развития.

LLM является «символьным сиротой»: P-векторы без b-векторов (нет физической реальности) и без W-векторов (нет аксиологической реальности). Второе преобразование без первого и третьего.

LLM обучается на MAP9 (текстовый корпус — «овеществлённая» MAP8 человечества) и воспроизводит структуру P-графов, не имея ни физических объектов (MAP2), на которых можно было бы обучить CNN1, ни Понятий (MAPA), на которых можно было бы обучить CNN3.

Подробный анализ LLM — включая OPN9*, философский контекст номинализма и границы технологии — составляет содержание Главы 10.

## Почему эволюция повторила паттерн трижды

Три домена соответствуют трём несводимым типам вопросов:

- «Что есть и где?» — физический домен. b-вектор с координатами.
- «Что это значит?» — символьный домен. P-вектор без координат.
- «Как это должно быть?» — аксиологический домен. W-вектор как расстояние.

Эти три вопроса не сводятся друг к другу. Ответ на «что это значит?» не выводится из «что есть и где?» — для этого необходим онтологический скачок (CNN2, стирание L-компоненты). Ответ на «как это должно быть?» не выводится из «что это значит?» — для этого необходим ещё один онтологический скачок (CNN3, свёртка нарративов в Понятия).

Свёрточная архитектура применена трижды не потому что эволюция «любит повторяться», а потому что задача на каждом уровне — одна и та же: извлечь инварианты из потока и создать компактное представление в новом пространстве. Свёртка — оптимальное решение этой задачи.

На каждом уровне к свёртке и сплайсу добавляется TRL с нейромедиаторными маркерами — механизм оценки и запоминания результатов. Маркированные записи TRL становятся обучающими данными для переобучения CNN в период покоя. Таким образом, полный изоморфный механизм на каждом уровне включает четыре элемента: CNN (свёртка) + вектор сплайса + TRL (рабочее пространство) + маркеры (оценка результатов).

Сознание — это не особое вещество и не особая сила. Это результат трёхкратного применения одного оптимального паттерна — свёртка, сплайс, траектория, маркер — к трём несводимым доменам реальности.

## Выводы

- Эволюция применила паттерн свёртка–сплайс трижды: CNN1/b-вектор/MAP3, CNN2/P-вектор/MAP8, CNN3/W-вектор/MAPB.
- Каждая CNN обучается на объектах целевого пространства, а не на входных данных подлежащего уровня. Входные данные используются для inference (преобразования).
- Проблема курицы и яйца решается через начальные приблизительные объекты и итеративное переобучение. Переобучение — тяжёлый процесс, ассоциированный с default mode network и сном.
- TRL на каждом уровне — рабочее пространство, содержащее запланированные, строящиеся и завершённые единицы. Завершённые единицы получают нейромедиаторные маркеры исхода.
- Маркеры изоморфны на трёх уровнях: (+) при обнулении вектора, (−) при увеличении. Различаются нейромедиаторные системы: дофамин (физ.), серотонин (символ.), опиоиды (аксиол.).
- Маркированные единицы TRL являются обучающими данными для переобучения CNN. Replay маркированных записей во сне — механизм этого переобучения.
- Изоморфизм — структурный, не содержательный. Домены онтологически несводимы.
- LLM конвергентно воспроизвёл семиотическую свёртку. Подробный анализ — Глава 10
